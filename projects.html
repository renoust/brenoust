Research Projects

Emergence by Design (MD)

European FP7 project: http://emergencebydesign.org
Emergence by Design studies innovation, and stability in an “innovation society”.
Partners: University of Warsaw, Euclid Network, FactLink, INRIA, Kennisland, ECLT, Chalmers University of Technology, TECH4i2, The HUB.

The TransMedia Observatory (OTMedia)

French ANR 2010CORD015: http://www.otmedia.fr
OTMedia aims to analyse the French news and media events from many different sources (internet, press, radio, tv).
Partners: INA, AFP, INRIA, LIA, CIM Sorbonne Nouvelle University, Syllabs, LATTS Marne-la-Vallée Paris-Est University - Ecole des Ponts Paritech.

Softwares

TulipPosy

http://tulipposy.labri.fr:31497
or on github Main contributor. TulipPosy is an online framework for graph visual analytics. The server side takes full advantages of the python interconnection of Tulip. The client side offers multiple views and interaction on top of d3.js.
User guide: PDF

Tulip

http://tulip.labri.fr Contributor. Tulip is a framework for huge graph analysis and visualization, with a lot of convenient plugins. It can be easily interfaced using its Python module. (This website background image has been generated using Tulip.)
